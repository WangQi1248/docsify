# 6.IDEA关闭和打开项目

- **关闭项目**

如果想关闭已有的项目，点击【File】【Close Project】

![1583648054534](assets/1583648054534.png)

可以看到下面的初始界面

![1583648211170](assets/1583648211170.png)

- 打开已有项目

如果左边没有打开过的项目记录，可以点击【Open】打开项目（注意一定具有idea的项目结构才行）
