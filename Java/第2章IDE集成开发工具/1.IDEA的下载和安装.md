# 1.IDEA的下载和安装
## 下载

下载地址  [https://www.jetbrains.com/idea/](https://www.jetbrains.com/idea/)  去找DOWNLOAD就可以了。

## 安装

在官网下载好安装包之后，双击打开安装向导，点击Next下一步。

![1583501951198](assets/1583501951198.png)

![1583502086734](assets/1583502086734.png)
